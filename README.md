## 🌳 사용 방법

### 빌드
```
yarn build
```

### 크롬 익스텐션에 추가
1. `chrome://extensions/` 접속
2. 개발자 모드 켜기
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. `dist` 폴더 선택
